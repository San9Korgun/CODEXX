# Друкарня онлайн

Цей репозиторій містить прототип простої вебсторінки для сервісу онлайн друку. 
Сайт натхненний можливостями сервісів, 
але пропонує більш сучасний і мінімалістичний інтерфейс.

## Запуск

Відкрийте `website/index.html` у браузері, щоб переглянути приклад сторінки.

## Структура

- `website/index.html` — основна сторінка з коротким описом послуг та прикладами продукції.
- `website/style.css` — стилі для оформлення сайту.

Це лише базовий шаблон, який можна розширювати згідно з вашими потребами.
